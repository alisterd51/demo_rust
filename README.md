# demo_rust

use rust compiler
```shell
rustc file.rs
```

use rust standard corrector
```shell
rustfmt file.rs
```

use cargo
```shell
cargo new project_name
```
```shell
cargo check
```
```shell
cargo build
```
```shell
cargo run
```
```shell
cargo clean
```
```shell
cargo doc --open
```
(for release)
```shell
cargo build --release
```
```shell
upx --brute target/release/project_name
```
